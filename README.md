# Proyecto: Pedido de Restaurante
Trabajo de Programación Orientada a Objetos (POO) en Java  
Autores: **Daniel Dos Santos Ferreira** y **Andrés Ramos Guerra**

----------------------

##  Descripción del proyecto

Este proyecto simula el sistema de pedidos de un restaurante utilizando conceptos de POO en Java:

Encapsulación  
Atributos estáticos  
Enumeraciones (`enum`)  
Validaciones  
Gestión de estados del pedido  
Trabajo colaborativo con Git y GitHub  

El programa permite:

1. Crear pedidos con un identificador único autogenerado  
2. Añadir hasta **5 artículos** por pedido  
3. Calcular automáticamente el **precio total**  
4. Cambiar el estado del pedido:
  `EN_PREPARACION`
  `LISTO_PARA_ENTREGAR`
  `ENTREGADO`
5. Evitar retrocesos de estado  
6. Mostrar un resumen completo del pedido

##  Cómo ejecutar el programa

1. Clona el repositorio
2. Abre la carpeta en Visual Studio Code o tu IDE favorito  
3. Compila las clases  
4. Ejecuta la clase **Main.java**

----------------------

##  Ejemplos incluidos

La clase `Main.java` contiene varios escenarios:

1. Pedido normal con varios artículos  
2. Cambio de estado correcto  
3. Intento de retroceder estado (no permitido)  
4. Intento de añadir más de 5 artículos
